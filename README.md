# UI - Structured users

### Clone the repository:
    
```
git clone https://github.com/fvjupiter/ui-structured-users.git
```

### Go to the directory:

```
cd ui-structured-users
```

### Install dependencies:
    
```
npm install 
```

### Run the application:
    
```
npm run dev
```

## Or visit the live demo at 
https://ui-structured-users.vercel.app
